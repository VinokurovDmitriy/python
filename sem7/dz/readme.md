### Телефонный справочник. Описание задания
Создать телефонный справочник с возможностью импорта и экспорта данных в нескольких форматах.
под форматами понимаем структуру файлов, например:в файле на одной строке хранится одна часть записи, пустая строка - разделитель
* Фамилия_1
* Имя_1
* Телефон_1
* Описание_1


* Фамилия_2
* Имя_2
* Телефон_2
* Описание_2

и т.д.в файле на одной строке хранится все записи,
символ разделитель - ;**
* Фамилия_1,Имя_1,Телефон_1,Описание_1
* Фамилия_2,Имя_2,Телефон_2,Описание_2
и т.д.
Предполагается возможность вывода всех контактов, поиска контакта по имени, добавления и удаления контакта

### Телефонный справочник. Описание функционала
* Для добавления записи в справочник, нужно заполнить необходимые поля и нажать кнопку сохранить.
* Для отображения записей в виде столбца нужно нажать кнопку Вид 1
* Для отображения записей в строку нужно нажать кнопку Вид 2
* Для поиска записей по имени нужно ввести имя в соответствующее поле и нажать кнопку Поиск напротив поля Имя
* Для поиска записей по фамилии нужно ввести фамилию в соответствующее поле и нажать кнопку Поиск напротив поля Фамилия
* Для поиска записей по описанию нужно ввести описание в соответствующее поле и нажать кнопку Поиск напротив поля Описание
* Для удаления записей по имени нужно ввести имя в соответствующее поле и нажать кнопку Удалить напротив поля Имя
* Для удаления записей по фамилии нужно ввести фамилию в соответствующее поле и нажать кнопку Удалить напротив поля Фамилия
* Для удаления записей по описанию нужно ввести описание в соответствующее поле и нажать кнопку Удалить напротив поля Описание
